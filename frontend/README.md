# QuerySafe Frontend

This folder contains the Next.js frontend for QuerySafe. It implements a demo UI including auth pages, a chat-style query interface, an explainability drawer, and an auto-dashboard generator.

Quick start:

```bash
cd frontend
npm install
npm run dev
```

API base URL: set `NEXT_PUBLIC_API_URL` in `.env.local` (defaults to `http://localhost/api`).

This implementation is a lightweight demo that follows the project's Frontend README. The app runs in demo mode without a backend by returning mock responses.
